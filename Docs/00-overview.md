# AgoraCommerce — Project Overview

## Goal
Build a portfolio-grade e-commerce backend demonstrating modern .NET engineering practices:
- Clean Architecture (Domain/Application/Infrastructure/API)
- MySQL + EF Core migrations
- Dockerized local development
- REST API with validation and authentication
- Unit + Integration + Architecture tests
- CI-ready structure

## Tech Stack
- .NET 8
- ASP.NET Core Web API
- EF Core
- MySQL 8
- Pomelo.EntityFrameworkCore.MySql
- FluentValidation
- Serilog
- Swagger / OpenAPI
- xUnit
- Docker Compose
